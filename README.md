# Treno na Ladeira
Este projeto em Java resolve um problema de Física sobre um trenó em plano inclinado com atrito. O programa calcula as forças necessárias para impedir o movimento, iniciar a subida e manter velocidade constante, validando entradas e exibindo resultados em Newtons.

# Funcionalidades
✅ Interface gráfica com Swing  
✅ Verificação e validação dos dados de entrada  
✅ Cálculo de forças em plano inclinado  
✅ Consideração de atrito estático e cinético  
✅ Exibição dos resultados em Newtons  
✅ Tratamento de erros com mensagens ao usuário
